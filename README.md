# 📊 GEO 商机工作台 — 360智汇商途

**GEO（生成式引擎优化）商机追踪与客户管理工具**

一键展示：https://liwei5666.github.io/geo-workbench/

---

## 这是什么

一个纯前端单页应用，运行在 GitHub Pages 上。

核心功能：把散落在各招标平台的 **GEO 项目** 汇总到一张表里，配合 **潜力客户挖掘**、**竞品分析**、**软件情报追踪**，帮 GEO 销售人员一站式管理商机。

---

## 功能模块

| 模块 | 说明 |
|------|------|
| **正在招标** | 当前可投标的 GEO 项目，含截止日期、招标平台、联系人，来源链接可溯源 |
| **已招标复联** | 已投/跟进中的客户，标记跟进状态和优先级 |
| **潜力客户** | 未覆盖的高价值潜客清单，含预算评估、GEO 需求描述、评分 |
| **竞品客户** | 分析其他 GEO 服务商在服务谁，找差异化切入点 |
| **软件洞察** | 追踪市面上的新 SaaS 产品，判断哪些厂商可能需要 GEO 服务 |

每条数据都附来源链接，信息可验证，不瞎编。

## 技术栈

- 纯 HTML + CSS + JavaScript
- Github Pages 托管
- 无后端、无数据库、开箱即用
- 数据内嵌在 `index.html` 的 `__DATA__` 对象中

---

## 更新方式

1. 修改 `index.html` 中的 `__DATA__` 数据段
2. `git add && git commit && git push`
3. GitHub Pages 自动部署，几分钟后生效
4. 也可直接在线编辑 GitHub 仓库文件
